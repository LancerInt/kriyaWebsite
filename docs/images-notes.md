# Image Placeholder Notes

Every image slot now points to a unique SVG placeholder so you can drop in final artwork without editing HTML. Swap each file with your final asset while keeping the same filename to preserve layout sizing across desktop, tablet, and mobile.

## Global sizing guidance
- **Square heroes:** 1200 × 1200 (1:1). Used for product hero visuals.
- **Wide content / packshots / mode visuals:** 1600 × 800 (2:1). Used for variant packs and mode-of-action illustrations.
- **Home & Garden packshots:** 900 × 1200 (3:4 portrait).
- **Existing brand/section placeholders:** 1500 × 1200 (5:4) for the Home & Garden hero; 1600 × 1200 (4:3) for corporate sections noted below.

## Usage reminders
- All placeholders use `object-fit: cover`; keep final imagery close to the noted ratios to avoid cropping.
- Replace the SVGs with optimized PNG/JPG/SVG files of similar dimensions for crisp results on high-density displays.
- Mobile and tablet views inherit these sizes—staying near the listed dimensions keeps layouts stable across breakpoints.
